！！新网点部署说明！！
   新网点部署，需要定义网点id（字母或数字），并修改cfg.yaml配置项nid.

cdrAlarm:       主程序
    参数：
        -as             运行模式：client||server||singleton。如下图。
        -cdrPath        网关模块cdr路径
        -alarmUri       告警Uri 
        -pushGateWayUri 监控Uri
cfg.yaml:         基本配置
    配置：
        -timeMinInterva     告警统计时间，分钟。
        -conAbnormal        连续故障告警阈值，如10，连续出现10个故障，即发出告警。
        -percentage         故障率告警阈值，如0.5，在timeMinInterva时间段内，故障率>50%，即发出告警。
        -nid                配置节点标识，如jsdx等。
run.sh:           运行脚本
./log/            运行日志


运行模式：【采集/告警分离：client/server】
+ ----------------------------------------------- +                 + ---------------------------- +

| 【cdr采集:client】 —:9098—> 【cdr告警：server】 |   —（公网）—>   | :9091【监控】 :9097【告警】  |

+ ----------------------------------------------- +                 + ---------------------------- + 
运行模式：【采集/告警一体：singleton】
+ ----------------------------------------------- +                 + ---------------------------- +

|      【cdr采集/cdr告警：client/server】         |   —（公网）—>   | :9091【监控】 :9097【告警】  |

+ ----------------------------------------------- +                 + ---------------------------- +
